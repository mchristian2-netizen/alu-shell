# Shell Basics Scripts
Each script performs a basic shell operation:
- 0-current_working_directory: prints absolute path of current directory
- 1-listit: lists contents of current directory
- 2-bring_me_home: changes directory to home
- 3-listfiles: lists in long format
- 4-listmorefiles: lists including hidden files
- 5-listfilesdigitonly: lists with numeric UID/GID
- 6-firstdirectory: creates /tmp/my_first_directory
- 7-movethatfile: moves betty file
- 8-firstdelete: deletes betty
- 9-firstdirdeletion: deletes /tmp/my_first_directory
- 10-back: goes to previous directory
- 11-lists: lists current, parent, and /boot in long format
- 12-file_type: prints type of /tmp/iamafile
- 13-symbolic_link: creates symbolic link __ls__ to /bin/ls
- 14-copy_html: copies newer .html files to parent
- 15-lets_move: moves uppercase-letter files to /tmp/u
- 16-clean_emacs: deletes files ending with ~
- 17-tree: creates nested directories welcome/to/school
